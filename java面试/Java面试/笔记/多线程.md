## 基本概念

并发：多个线程同时运行，如果是单核，那么需要切入和切出。如果是多核，那么就是真正的多线程在跑。
多个线程操作相同的资源，保证线程安全，合理使用资源。

同时处理很多请求，提高程序的性能。
