# 概述
面向切面编程；其实就是面向特定方法编程；
场景：
例如系统中有部分业务非常缓慢，需要定位执行时间较长的业务方法；此时需要统计每个业务的执行耗时；

实现：
通过动态代理技术实现；是最主流的实现方式；

# AOP进阶
## 通知类型
@Around 环绕通知
@Before 前置通知
@After 后置通知，无论是否有异常，都会通知
@AfterReturning 返回后通知
@AfterThrowing 异常后通知
